Sincronização com MillDesk

Hoje a ponte é manual. Assim que a API estiver confirmada, plugamos aqui.

Status atual

Manual (CSV) — os chamados vivem neste arquivo e podem ser exportados/importados em CSV para ida e volta com o MillDesk.

Onde ficam salvas as edições

Este arquivo é estático (roda direto no navegador, sem servidor). As alterações que você fizer aqui (novo chamado, edição, exclusão) ficam salvas no armazenamento local do navegador de quem estiver usando — não sincronizam automaticamente entre pessoas ou dispositivos. Para compartilhar o estado atualizado com o time, use Exportar CSV e distribua o arquivo, ou suba a versão exportada de volta ao repositório.

Para habilitar a sincronização automática com a API

  1. Confirme com o suporte do MillDesk se existe API REST disponível no seu plano.
  2. Solicite documentação e uma credencial de integração.
  3. Traga essas informações de volta — a sincronização automática pode ser adicionada sem refazer esta ferramenta.

Acesso

Login e senha para abrir esta ferramenta. Como é um arquivo sem servidor, essa senha protege contra acesso casual, mas não é segurança de verdade — qualquer pessoa com acesso ao arquivo e conhecimento técnico pode contornar. Não use para dados sigilosos sem uma solução de hospedagem com autenticação real.
